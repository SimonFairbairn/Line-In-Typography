Line In Typography

An Example Page

Javascript


/*
 *  Variables:
 *
 *  containers are what the plugin is attached to
 *
 *  fontSizeContainer (default: body): 	Whichever element explicitly declares the font size and
 *					Line Height (#wrap in this example)
 *  backgroundOffset (default: 0):	Just in case you need to offset the lines
 *
 */
jQuery(document).ready(function($) {
	var myOptions = {   
		lineHeightContainer : 'body',
		gridState           : 'on',
		lineState           : 'off',
		testHtml            : "<p>My Paragraph Test</p>",
		backgroundOffset    : -20
	};
	$( '#wrap' ).LineInTypography(myOptions);
});

CSS

The above settings would work well on a document that had the following CSS: 

body {
	font-size: 100%;	
}

#wrap {
	font-size: 1em; /* Default 16px in browser */
	line-height: 1.5;
}
